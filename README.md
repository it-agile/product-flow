# HUGO als Static Site Generator
Die Webseite benutzt HUGO als Static Site Generator. HUGO generiert die Webseite ins public-Verzeichnis.

In Github ist eie Workflow definiert, der bei Änderungen automatisch die Seite mit HUGO neu baut.

Um Änderungen lokal auf dem eigenen Rechner testen zu können, muss man HUGO installiert haben.

# Verzeichnisse
Die HTML-Templates liegen im layout-Verzeichnis. Der Content im Markdown-Format liegt im content-Verzeichnis.

Im static-Verzeichnis liegen statische Ressourcen wie CSS, Bilder, JavaScript, die durch HUGO nicht verändert werden und einfach unverändert in das public-Verzeichnis kopiert werden.
